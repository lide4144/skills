# 动态分析框架

## 证据单元

用“文本动作 → 读者状态变化 → 后续回报”记录关键节点。为判断区分文本事实、多个信号支持的强推断，以及存在同等替代解释的弱推断。

## 信息操控

追踪谁知道什么、何时知道、读者比角色多知道还是少知道。观察伏笔的可见度与兑现距离、视角切换制造的危险预告或认知差、规则是否在解题前公平展示、悬念属于“发生什么/为何发生/如何解决”哪一种，以及揭晓是否改变旧信息的意义。

## 冲突转译

按需要区分三层，不强求都有：表层阻碍；身份、资源、制度与能力边界构成的规则层；角色认为什么正当、值得牺牲的价值层。重点分析抽象矛盾如何落到一个可见选择、物件、期限、公开场合或不可逆代价。检查双方是否都能施压、升级是否改变局面、胜负是否产生后果。

## 情绪波形

用带触发点的序列表达，例如：`安稳（目标可控） → 不安（异常信息） → 憋屈（行动受限） → 期待（反制线索） → 释放（公开兑现） → 余震（新代价）`。

同时检查压力来源与增幅、释放兑现哪笔承诺、爽点是否在能力/资源/地位/认知/关系/道德确认/探索发现间轮换、日常段落是否恢复情绪或暗埋信息、章末钩子是否因重复而疲劳。

## 心流与共鸣

寻找读者进入角色处境的具体接口：身体微反应、受限选择、社会性羞耻、具体损失、未说出口的话、熟悉的欲望。识别复合情绪，如庆幸夹杂羞耻、胜利后的空虚、算计成功的默契；说明它由哪些冲突信号构成。

## 网文结构单元

- 金手指：解决信息、资源、行动还是身份困境；有无选择成本；主动开发还是被动发放；是否挤压配角和悬念。
- 打脸/震惊：是否先建立判断标准与见证场，再形成合理误判，用可验证结果翻转地位，最后展示关系后果。
- 降维打击：优势是否提前展示，对手的自信是否符合其信息，胜利是否来自已建立能力。
- 升级：数值增长是否带来新选择与冲突，而非只把旧敌人换皮。
- 换地图：旧承诺是否收束，新地图是否改变规则、关系与冲突模式。
- 副本/循环：进入目标、局部规则、试错代价、破局信息与退出余波是否闭环。

## 反事实检验

对核心判断问：“如果删掉、前移、后移或换视角，会怎样？”用它确认技法的真实功能。建议必须指出改哪一段、压缩什么、补哪类信号、把兑现移动到哪里，以及预期改变哪条情绪曲线；不要把作品改成分析者偏爱的另一类型。

## 常见误判

- 世界观信息多，不等于信息操控高明。
- 角色吵得激烈，不等于价值冲突深。
- 三章一个事件，不等于三章一个爽点。
- 主角获胜，不等于情绪充分释放。
- 配角惊讶，不等于读者也会惊讶。
- 伏笔被解释，不等于兑现令人满足。
- 慢节奏不必然拖沓；快节奏也可能没有积累。
