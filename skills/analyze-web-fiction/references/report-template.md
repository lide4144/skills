# 报告组织建议

模板用于保证证据与层次，不要求逐栏填满。

## 分析范围与结论边界

说明是完整文本还是取样，列出覆盖位置、可能遗漏和正文清洗情况。用 2–4 句概括主要阅读承诺与创作发动机，不写长摘要。

## 核心洞见

用 3–6 个带结论的标题展开。每节尽量包含关键证据或位置、叙事选择、读者认知/情绪变化、收益与代价，以及必要时的最小修改建议。从信息操控、冲突转译、情绪波形、共鸣接口和网文结构单元中择优，不机械各写一节。

整书深度报告必须另设三个短节或融入核心洞见：设定新颖性（说明基线与因果作用）、情感显微镜（分析具体微反应链与关系余波）、文以载道（展示价值如何由选择、代价和后果承载）。

## 情绪与兑现

给出带触发点的情绪曲线。整书报告还需讨论爽点轮换、承诺回收速度、低谷长度和章末钩子重复度。

## 有效性诊断

分别指出最有效的机制、最明显的断裂（铺垫/升级/兑现/余波），以及一项写清修改对象与预期效果的高优先级修改。

## 创作手法总评

用一句话提炼创作发动机，再给出“最值得学习的一招”和“最该警惕的一处”。除非用户要求，不用分数制造虚假精确度。
