# 长文本分章与分块流程

1. 运行 `python scripts/split_novel.py <原文.txt> --output <目录> --chunk-chars 80000`。
2. 读取 `manifest.json`，检查章节数、标题、未匹配前缀、各块章节范围和超长章节标记。
3. 章节数异常时用 `--chapter-pattern` 指定正则，并先用 `--dry-run` 验证。
4. 按 `chunks/0001.md` 起顺序分析。笔记保留章节坐标、承诺与兑现、情感余波、设定新信息和价值选择。
5. 每 8–12 块做阶段归纳；最终综合后回查原块验证关键结论和至少一个反例。

分块优先遵循章节边界。单章超长时才在段落边界切分并在 manifest 标记。不要把字符块边界误当成叙事边界。
